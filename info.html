Screening non clinico open-source per tratti autistici e ADHD negli adulti. Completamente gratuito, senza tracciamento, senza raccolta dati.

Non-clinical open-source screening for autistic traits and ADHD in adults. Completely free, no tracking, no data collection.

Come eseguire NeuroScreen in locale

How to run NeuroScreen locally

I questionari e i task CPT / Sociale funzionano anche aprendo direttamente il file index.html. La webcam richiede un server HTTP locale (i browser bloccano la webcam su file://).

Questionnaires and CPT / Social tasks also work by opening index.html directly. The webcam requires a local HTTP server (browsers block webcam on file://).

Windows — PowerShell (inclusobuilt-in):

.\start.ps1
→ poi aprithen open http://localhost:8080

Python 3 (qualsiasi sistemaany system):

python -m http.server 8080

Node.js:

npx serve .

È incluso vercel.json con header di sicurezza preconfigurati (Permissions-Policy: camera=(self), ecc.). Per Vercel: vercel --prod.

Privacy

Nessun dato raccolto

No data collected

⚠️ Nessun dato viene mai inviato a server remoti.
Tutta l'elaborazione avviene localmente nel browser. La webcam non viene mai registrata o trasmessa. I risultati vengono salvati esclusivamente in localStorage del browser per la funzione "ripristina sessione" (TTL 7 giorni, cancellabile con "Ignora").
⚠️ No data is ever sent to remote servers.
All processing happens locally in the browser. The webcam is never recorded or transmitted. Results are saved exclusively in the browser's localStorage for the "restore session" function (TTL 7 days, clearable with "Dismiss").

NeuroScreen non usa cookie, analytics, tracking o pubblicità di alcun tipo. Il codice è aperto e verificabile su GitHub.

NeuroScreen uses no cookies, analytics, tracking or advertising of any kind. The code is open and verifiable on GitHub.

Limiti dello strumento

Tool limitations

Questo strumento non è un test diagnostico. I risultati sono indicativi e non sostituiscono la valutazione di un neuropsichiatra o psicologo clinico. Per una diagnosi formale di ASD o ADHD è necessaria una valutazione neuropsicologica completa.

Per i dettagli su sensibilità, specificità e soglie cliniche di ogni strumento, consulta la sezione Docs ↗.
This tool is not a diagnostic test. Results are indicative and do not replace evaluation by a neuropsychiatrist or clinical psychologist. A full neuropsychological assessment is required for a formal ASD or ADHD diagnosis.

For details on sensitivity, specificity and clinical thresholds for each tool, see the Docs ↗ section.
